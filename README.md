# 🌦️ Weather Web Application

## 📖 Overview

The Weather Web Application is a simple and interactive web project developed to provide real-time weather updates for different cities around the world. Users can search for any city and view weather details such as temperature, humidity, wind speed, and weather conditions. The application also includes a 5-day weather forecast feature for better weather analysis.

---

## ✨ Features

* 🌍 Real-time weather updates
* 🔍 Search weather by city name
* 📅 5-day weather forecast
* 📱 Responsive and user-friendly interface
* 🌡️ Display of temperature, humidity, and wind speed
* ⚡ Dynamic weather data using API integration

---

## 🔗 API Used

The project uses the OpenWeatherMap API to fetch real-time weather and forecast data for different cities and locations.

---

## ⚙️ Working of the Project

The application takes a city name as input from the user and sends a request to the weather API using JavaScript. The API returns weather information, which is then displayed dynamically on the webpage through an interactive interface.
